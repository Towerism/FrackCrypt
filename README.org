* FrackCrypt
Encryption Back Ends
** Requirements
- CMake 3.0+
** Instructions
*** Building
Generate the Makefiles, and then run Make:
#+BEGIN_SRC
$ mkdir -p build && cd build
$ cmake ..
$ make
#+END_SRC
To run tests, use the ~FRACKCRYPT_TEST~ option when configuring, i.e.:
#+BEGIN_SRC
$ cmake -DFRACKCRYPT_TEST=ON ..
$ make run-tests
#+END_SRC
